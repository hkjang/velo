Quick Start
5분 만에 시작하기
단일 JAR 실행으로 WAS + AI 플랫폼 + 에이전트 허브를 즉시 가동
# 1. Download & Run
java -jar was-bootstrap-0.6.0-jar-with-dependencies.jar

# 2. OpenAI-Compatible API
curl -X POST http://localhost:8080/ai-platform/v1/chat/completions \
  -H "Authorization: Bearer velo-demo-key" \
  -H "Content-Type: application/json" \
  -d '{"messages":[{"role":"user","content":"Hello"}]}'

# 3. A2A Agent-to-Agent (JSON-RPC 2.0)
curl -X POST http://localhost:8080/ai-platform/a2a \
  -d '{"jsonrpc":"2.0","method":"tasks/send","params":{"message":{"role":"user","parts":[{"type":"text","text":"번역해주세요"}]},"capability":"translation"},"id":1}'

# 4. Agent Card Discovery
curl http://localhost:8080/ai-platform/.well-known/agent.json

# 5. Dashboard
open http://localhost:8080/ai-platform
# Python (OpenAI SDK)
from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:8080/ai-platform/v1",
    api_key="velo-demo-key"
)

response = client.chat.completions.create(
    model="llm-general",
    messages=[{"role": "user", "content": "Hello"}]
)
